# 📞 Teleperformance Call Center Performance Dashboard

An end-to-end Power BI dashboard designed to monitor and analyze Call Center performance using real-world business KPIs.

The project focuses on transforming raw operational data into actionable insights that support better decision-making for managers and business stakeholders.

---

# 📌 Project Overview

This dashboard provides an interactive overview of call center operations, enabling users to monitor service quality, agent performance, operational efficiency, and forecasting accuracy.

The solution was developed following Business Intelligence best practices using Power BI, Power Query, DAX, and Star Schema modeling.

---

# 🎯 Business Objectives

- Monitor overall Call Center performance.
- Track Service Level Agreement (SLA) compliance.
- Measure operational efficiency.
- Evaluate Agent Performance.
- Compare Forecasted Calls vs Actual Calls.
- Identify performance trends over time.
- Generate executive-level business insights and recommendations.

---

# 📊 Key Performance Indicators (KPIs)

- 📞 Total Calls Offered
- ☎️ Total Calls Handled
- ❌ Total Calls Abandoned
- 🎯 Service Level
- ⚡ Handling Rate
- 🚀 Efficiency
- ⏱ Average Speed of Answer (ASA)
- 📈 Forecast Accuracy
